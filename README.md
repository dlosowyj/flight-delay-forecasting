# flight-delay-forecasting
Attemps to classify flights as delayed or on-time based on historical data.

## flight_delay_eda_model_testing.ipynb
Contains exploratory data analysis and model testing on PDX flight delays from 2021-2025.

## flight_delay_rf_classifier.ipynb
Trains and tests a random forest model to classify flight delays from the 33 busiest airports in the US.

## flight_delay_web_scrape.ipynb
Selenium-based web scraping tool to gather data from Bureau of Transportation Statistics for the 33 busiest airports in the US.
